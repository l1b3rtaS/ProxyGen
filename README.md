# ProxyGen


Install ProxyGen
----------
```
git clone https://github.com/l1b3rtaS/ProxyGen
```
Install Python Modules
----------
```
pip install colorama
pip install tqdm
```
Launch
----------
```
cd ProxyGen
python3 ProxyGen.py
```

How it work
----------
```
You choose port and ip pattern. Next, list.txt will appear in the ProxyGen folder
```
